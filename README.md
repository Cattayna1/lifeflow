# Lifeflow

## Project Overview
Lifeflow is an innovative application designed to help users manage their daily activities, track progress, and cultivate positive habits. It integrates various tools and features to create a cohesive workflow that enhances productivity and personal growth.

## Vision
Our vision is to empower users to take control of their lives by providing a comprehensive workflow management tool that adapts to individual needs and promotes lifelong learning.

## Features
- **Task Management:** Create, edit, and delete tasks with deadlines.
- **Progress Tracking:** Visualize progress through charts and statistics.
- **Habit Building:** Set and track daily habits to promote positive behavior.
- **Reminders:** Receive notifications for upcoming tasks and habits.
- **User-Friendly Interface:** Intuitive design making it easy to navigate.

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js with Express
- **Database:** MongoDB
- **Deployment:** AWS
- **Version Control:** Git

## Getting Started
### Prerequisites
- Node.js installed on your machine.
- MongoDB setup and running.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Cattayna1/lifeflow.git
   cd lifeflow
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Set up your MongoDB URI in `.env`:
   ```bash
   MONGODB_URI=your_mongodb_uri_here
   ```
4. Start the application:
   ```bash
   npm start
   ```
5. Open your browser and navigate to `http://localhost:3000` to access Lifeflow.
